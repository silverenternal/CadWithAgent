# CadAgent AI 自主化优化建议

**创建日期**: 2026-04-06 | **阅读时间**: 10 分钟

---

## 🎯 核心结论

### 当前定位
**CadAgent = 几何增强的 VLM 助手** (单轮推理 + 工具调用)

### 目标定位
**CadAgent = 自主 CAD 设计代理** (多轮规划 + 自我反思 + 知识沉淀)

---

## 📊 能力差距总览

| 能力层 | 当前状态 | 目标状态 | 优先级 |
|--------|---------|---------|--------|
| 📁 图纸解析 | ✅ 2D (SVG/DXF) | ✅ 3D (STEP/IGES/B-Rep) | 🔥 P0 |
| 🧠 几何理解 | ✅ 基元 + 关系 | ✅ 3D 特征 + 设计意图 | 🔥 P0 |
| 🛠️ 设计优化 | ⚠️ 冲突检测 + 建议 | ✅ 自动参数化 + 生成式 | 🔥 P0 |
| 🤖 自主决策 | ⚠️ 单轮 LLM | ✅ 多轮规划 + 反思 | 🔥 P0 |
| 💾 记忆系统 | ⚠️ LRU 缓存 | ✅ 知识库 + 用户偏好 | P1 |

## 💡 关键设计决策

### 决策 1: STEP 解析方案

| 选项 | 优点 | 缺点 | 推荐 |
|------|------|------|------|
| `stepnc-rs` | 纯 Rust, 轻量 | 功能有限 | ⭐⭐ |
| `opencascade-sys` | 功能完整 | C++ 依赖大 | ⭐⭐⭐ |

**推荐**: 先用 `opencascade-sys` 快速验证，后期替换核心算法

---

### 决策 2: LLM 部署方案

| 选项 | 优点 | 缺点 | 推荐场景 |
|------|------|------|---------|
| 云端 API (Qwen/GPT) | 能力强 | 延迟高，隐私 | 复杂任务 |
| 本地模型 (7B/14B) | 低延迟，隐私 | 能力弱 | 简单任务 |

**推荐**: 混合模式 - 路由策略
```rust
match task_complexity {
    High => use_cloud_llm(),
    Medium => try_local_first(),
    Low => use_local_llm(),
}
```

---

### 决策 3: 约束求解方案

| 选项 | 优点 | 缺点 |
|------|------|------|
| 自研求解器 | 可控，轻量 | 开发周期长 |
| 集成现有库 | 快速验证 | 依赖复杂 |

**推荐**: 集成 `petgraph` + 自研牛顿法求解器

## ⚠️ 风险提示

### 高风险项

| 风险 | 概率 | 影响 | 缓解措施 |
|------|------|------|---------|
| STEP 解析复杂度超预期 | 🔴 中 | 🔴 高 | 早期验证 + 成熟库 |
| 约束求解收敛困难 | 🔴 高 | 🔴 高 | 混合方法 + 超时 |
| LLM 延迟过高 | 🟡 中 | 🟡 中 | 本地模型 + 缓存 |

### 应对策略

1. **每周进度审查**: 周五下午检查里程碑进度
2. **早期验证**: 每阶段第 1 周完成技术验证
